## Linux Screen command reference

Create a new window : Ctl+a c
List windows: `Ctl+a "`
Switch window: `Ctl+a N`, for number N
Split horizontal:  `Ctl+a S`
Split vertical: `Ctl+a |`
Switch focus: `Ctl+a tab`
Close region: `Ctl+a X`
Detach: `Ctl+a d`
Reattach: `$ screen -r`
